# Лабораторная работа №2: NLP (Задание на тройку)

## Описание
Данная работа посвящена развертыванию локального сервера **Ollama** с легковесной языковой моделью **Qwen2.5:0.5B** и реализации программного интерфейса для взаимодействия с ней. Основной упор сделан на организацию автоматизированного процесса инференса через HTTP API и сохранение данных в структурированный отчет.

## Стек технологий
* **Ollama**: Среда для локального запуска и управления LLM.
* **Qwen2.5:0.5B**: Компактная языковая модель.
* **Python 3.x**: Язык реализации клиентской части.
* **Requests**: Библиотека для взаимодействия с REST API Ollama.
* **CSV**: Формат хранения итогового отчета.

## Технические требования
В ходе выполнения работы были соблюдены следующие условия:
1. **HTTP Инференс**: Взаимодействие с моделью происходит через запрос к локальному серверу Ollama.
2. **Автоматизация**: Написан скрипт на Python, который последовательно обрабатывает список промптов.
3. **Документирование**: Все функции скрипта содержат полноценную **Docstring** документацию.
4. **Формирование отчета**: Результаты работы сохраняются в файл `inference_report.csv` с кодировкой `utf-8-sig` для корректного отображения кириллицы.

## Установка и запуск

1. **Развертывание Ollama**
   Установите Ollama с официального сайта или выполните команду:
   ```bash
   curl -fsSL [https://ollama.com/install.sh](https://ollama.com/install.sh)
   ```

2. **Подготовка модели**
   Загрузите необходимую модель:
   ```bash
   ollama pull qwen2.5:0.5b
   ```

3. **Запуск API сервера**
   Убедитесь, что сервер запущен и принимает запросы (по умолчанию на порту 11434):
   ```bash
   ollama serve
   ```

4. **Запуск скрипта**
   Установите зависимости и выполните код:
   ```bash
   pip install requests
   python main.py
   ```

## Структура отчета (inference_report.csv)

Итоговый файл содержит таблицу со следующими столбцами:
* **запрос**: Исходный текст вопроса, отправленный модели.
* **ответ**: Сгенерированный текст ответа от LLM.

*Ввиду малого объема параметров используемой модели (0.5B), ответы могут содержать фактические ошибки или галлюцинации, что учитывается в рамках данного исследования.*
